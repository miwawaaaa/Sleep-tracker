# Sleep-tracker
A service for monitoring and improving sleep through tracking, analytics, and the development of healthy evening habits. 
# Проблема
Багато підлітків і дорослих мають труднощі із засинанням та нерегулярний режим дня, що негативно впливає на самопочуття. Цей вебзастосунок допоможе користувачам відстежувати сон, формувати здорові вечірні звички та отримувати просту аналітику для покращення якості відпочинку.
# MVP
Ключовий сценарій:
1. Користувач вводить або фіксує час засинання та пробудження;
2. Система зберігає ці дані;
3. Користувач бачить запис у своєму особистому списку або календарі;
4. Сервіс автоматично обчислює тривалість сну та відображає базову статистику (наприклад, середню тривалість за тиждень).
# Стейкхолдери
- Користувачі (підлітки та дорослі):
Потребують простого та зрозумілого інтерфейсу для швидкого внесення даних про сон та перегляду базової статистики.
- Власник / Автор (студент):
Мета — перевірити гіпотезу, що компактний та зручний застосунок допомагає користувачам контролювати сон, з урахуванням обмеженого часу та ресурсів.
- Викладач / Оцінювач:
Слідкує за відповідністю MVP вимогам, оцінює реалізацію головного сценарію та наявність чітких, вимірюваних показників успіху.
# Мета першої ітерації
Метою першої ітерації є створення стабільного вебзастосунку, який дозволяє користувачу:
1. швидко фіксувати час засинання та пробудження;
2. переглядати історію записів;
3. отримувати базову статистику тривалості сну.
Продукт повинен бути інтуїтивно зрозумілим і не вимагати складних налаштувань.
# Ознаки успіху першої ітерації
1. Користувач здатний виконати головний сценарій без додаткових пояснень і зайвих кроків;
2. Дані зберігаються коректно та відображаються без технічних помилок протягом щонайменше одного тижня використання.
# Користувацький сценарій
1. Користувач відкриває вебзастосунок у браузері.
2. Він уже має створений обліковий запис і увійшов у систему.
3. На головній сторінці доступна можливість додати новий запис про сон.
4. Дані за попередні дні зберігаються та відображаються в особистому кабінеті.
