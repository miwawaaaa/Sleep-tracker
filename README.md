# Sleep-tracker
A service for monitoring and improving sleep through tracking, analytics, and the development of healthy evening habits. 
# Проблема
A Багато підлітків і дорослих мають труднощі із засинанням та нерегулярний режим дня, що негативно впливає на самопочуття. Цей вебзастосунок допоможе користувачам відстежувати сон, формувати здорові вечірні звички та отримувати просту аналітику для покращення якості відпочинку.
